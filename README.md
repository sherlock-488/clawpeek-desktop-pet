# ClawPeek Desktop Pet

This is the GitHub-friendly repository root for ClawPeek Desktop Pet.

If you create a GitHub repository for this project, use the name `clawpeek-desktop-pet`.

## Structure

- `package.json`: root wrapper scripts
- `clawpeek-desktop-pet-v0.4.0/`: actual Electron app source

## Commands

Run these from the repository root:

```powershell
npm run install:app
npm start
npm test
```

## Notes

- The main project README is in [`clawpeek-desktop-pet-v0.4.0/README.md`](./clawpeek-desktop-pet-v0.4.0/README.md).
- `node_modules` is ignored and should not be committed.
- The current nested layout is usable for GitHub as-is.
